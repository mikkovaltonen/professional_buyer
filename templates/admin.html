Professional Buyer Admin Documentation & System Overview

Agentic Workflow & Tools

This diagram shows the complete agentic workflow and tool integration for the Professional Buyer system.

Agent Prompts & Configuration
SearchAgent

Purpose: Real-time web search for market data and external supplier research

Instructions: "You immediately provide an input to the WebSearchTool to find up-to-date information on the user's query. Log: 'SearchAgent activated for real-time web search'"

InternalKnowledgeSearchAgent

Purpose: Search internal procurement data and documents

Instructions: Searches internal vector store for:

  • Toimittaja katalogit (supplier catalogs)
  • Kilpailutetut sopimukset (competitive contracts)
  • Kilpailutetut hinnat (competitive pricing)
  • Tuotteet (products)
ApprovalSpecialistAgent

Purpose: Handle purchase order approvals and notifications

Instructions: Responsible for approval workflows, authorization processes, and stakeholder communication

POPostingAgent

Purpose: Post purchase orders to ERP system

Instructions: Handles PO creation and posting with supplier, product, and pricing data

ProfessionalBuyerAssistant (Triage Agent)

Purpose: Main orchestrator that routes requests to specialist agents

Core Benefits Focus:

  1. Save money by aligning with negotiated contracts
  2. Save fixed fees from internal procurement services
  3. Reallocate professional buyer time from operational transactions to strategic savings
Available Tools & Functions
WebSearchTool
Used by SearchAgent
Real-time web search for market trends, supplier information, and competitive intelligence
FileSearchTool
Used by InternalKnowledgeSearchAgent
Search internal vector store (ID: configured via VECTOR_STORE_ID env var)
request_po_approval(po_number, amount, reason)
Used by ApprovalSpecialistAgent
Request approval for purchase orders with automatic routing based on amount thresholds
send_email(recipient, subject, message)
Used by ApprovalSpecialistAgent
Send email notifications to stakeholders and approvers
po_posting_api(toimittaja, tuote, hinta)
Used by POPostingAgent
Post purchase orders to ERP system with supplier, product, and pricing information
